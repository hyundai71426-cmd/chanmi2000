====================================================
  찬미예수 2000 악보 웹사이트 사용 방법
====================================================

[파일 구성]
  index.html   - 웹사이트 메인 파일
  prepare.py   - 데이터 준비 스크립트 (한 번만 실행)
  songs.json   - prepare.py 실행 후 자동 생성
  images/      - prepare.py 실행 후 자동 생성 (PNG 악보 이미지)

----------------------------------------------------
[STEP 1] Python 스크립트 실행 (한 번만)
----------------------------------------------------
1. 윈도우 시작 → "명령 프롬프트" 검색 후 실행
2. 아래 명령어 입력:

   cd /d D:\테스트\chanmi2000
   python prepare.py

3. 완료되면 images/ 폴더와 songs.json 파일이 생성됩니다.
   (BMP → PNG 변환으로 파일 크기가 훨씬 작아집니다)

* Python이 없으면: https://python.org 에서 설치
* 설치 시 "Add to PATH" 반드시 체크!

----------------------------------------------------
[STEP 2] 인터넷에 올리기 (GitHub Pages 무료 사용)
----------------------------------------------------
1. https://github.com 회원가입 (무료)

2. 새 저장소(Repository) 만들기:
   - Repository name: chanmi2000-scores (또는 원하는 이름)
   - Public 선택
   - Create repository 클릭

3. 파일 업로드:
   - "uploading an existing file" 클릭
   - 아래 파일/폴더를 모두 끌어다 놓기:
       index.html
       songs.json
       images/ 폴더 전체
   - Commit changes 클릭

4. GitHub Pages 활성화:
   - 저장소 → Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: main / (root)
   - Save 클릭

5. 약 1~2분 후 아래 주소로 접속 가능:
   https://[내 GitHub ID].github.io/chanmi2000-scores/

----------------------------------------------------
[웹사이트 기능]
----------------------------------------------------
  - 번호로 검색: 1889, 1890 등 숫자 입력
  - 제목으로 검색: 찬양 제목 검색
  - 가사로 검색: songs.json에 가사가 있는 경우
  - 이전/다음 버튼으로 악보 넘기기
  - 키보드 ← → 화살표로도 이동 가능
  - ESC 키로 악보 닫기
